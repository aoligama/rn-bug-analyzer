# React Native Bug Analyzer

A VSCode extension for analyzing React Native errors and providing solutions.

## Features

- Analyze React Native error messages
- Get detailed solutions and recommendations
- Support for common React Native error patterns
- Easy-to-use interface

## Usage

1. Select error text or open the analyzer
2. Use Command Palette and search for "RN: Analyze Error"
3. Or use the keyboard shortcut: `Ctrl+Shift+A` (Windows/Linux) or `Cmd+Shift+A` (Mac)

## Requirements

- VSCode 1.80.0 or higher

## Extension Settings

This extension contributes the following settings:

- `rn-bug-analyzer.enable`: Enable/disable this extension
